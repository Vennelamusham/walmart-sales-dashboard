# Walmart Sales Analysis Dashboard
Overview
This project analyzes Walmart retail sales data using Power BI to generate actionable business insights.

Key Features
- Interactive dashboard with slicers (Store, Date, Holiday)
- KPI metrics: Total Sales, Average Sales, Max & Min Sales
- Sales trend analysis over time
- Store-wise performance comparison
- Impact of holidays on sales
- Analysis of external factors like fuel price and unemployment

Tools Used
- Power BI
- DAX
- Power Query
- Excel (CSV)

Insights
- Sales vary significantly across stores and time
- Holiday periods show noticeable impact on sales
- External economic factors influence sales performance
